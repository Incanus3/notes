## Rails gems / integrations

- https://github.com/thomas-mcdonald/bootstrap-sass (last update 01/2026) - Sass port of Bootstrap 2 and 3; more commonly used than the Less version.
- https://github.com/seyhunak/twitter-bootstrap-rails (last update 02/2026) - Bootstrap integration for Rails that uses Less.
- https://github.com/potenza/bootstrap_form (last update 05/2026) - Rails form builder for Bootstrap.
- https://github.com/plataformatec/simple_form (last update 04/2026) - Rails form builder DSL with no strong opinion on markup.
- https://github.com/rafaelfranca/simple_form-bootstrap (last update 04/2026) - example application using Simple Form and Bootstrap with code examples.
- http://simple-form-bootstrap.plataformatec.com.br/articles/new - Simple Form + Bootstrap example application article.
- https://github.com/silviomoreto/bootstrap-select (last update 07/2024) - jQuery plugin for enhanced Bootstrap select elements.
- https://github.com/twg/active_link_to (last update 04/2024) - Rails helper that adds an active class to links or enclosing tags.

## Bootstrap quirks

- Empty description-list values (`<dd>`) in Bootstrap `.dl-horizontal` can cause following descriptions to shift upward. Use `<dd class="clearfix">`.
- http://stackoverflow.com/questions/10566451/prevent-twitter-bootstrap-empty-dd-filling-with-next-dd-value - StackOverflow discussion about preventing empty Bootstrap `<dd>` values from being filled by the next value.
