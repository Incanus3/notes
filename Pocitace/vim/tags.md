* g] - go to definition (using ctags) - :tselect, C-] - :tag, g C-] - :tjump
* :tselect - list matching tags (:sts - with split)
* :tag {ident} - jump to definition of ident - C-] for current keyword
* :tjump - like :tselect, but jump directly, if only one match - g C-] (:stj)
* :tn, :tp - next/previous match
* :tf[irst], :tl[ast]
* C-w ] - split window and jump to tag
* prefix with p to use preview window
