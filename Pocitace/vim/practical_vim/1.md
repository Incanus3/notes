* change this
''var foo = "method("+argument1+","+argument2+")";''
* to this
''var foo = "method(" + argument1 + "," + argument2 + ")";''
• f+ - jump to plus
• s_+_<Esc> - replace + by _+_ (s = cl)
• ; - repeat find
• . - repeat action
• ;.;. - again and again

* change several occurrences of word under cursor
* * - start search for word under cursor
* cw<changes><Esc> - change word to <changes>
* n.n. - repeat for subsequent occurrences
