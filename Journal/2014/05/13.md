== Bigint in Rails migrations ==
https://moeffju.net/blog/using-bigint-columns-in-rails-migrations
- use limit: 8 to get bigint column
- use this to get bigint id column:
class CreateDemo < ActiveRecord::Migration
  def self.up
    create_table :demo, :id => false do |t|
      t.integer :id, :limit => 8
    end
  end
end
