== Šablona kurzu (product) ==
- editovat na backendu
  - název (name) - string
  - číslo produktu/receptury/sklad. karty (card_number) - integer
  - kategorie (category) - string
  - kapacita  (capacity) - integer
  - max. kapacita (max_capacity) - integer
  - cena (price) - numeric(12,2)
  - cena 2 (price2) - numeric(12,2)
  - daň. sazba (tax_rate) - numeric(4,2)
  - daň. sazba 2 (tax_rate2) - numeric(4,2)
- editovat na frontendu
  - HTML popisek (description) - HTML string
  - obrázky (images) - array<integer foreign key>
  - podobné kurzy (similar) - array<integer foreign key>

== Kurz (event) ==
- na backendu
  - začátek (start_at) - timestamp
  - konec (end_at) - timestamp
  - šablona (product) - integer foreign key
  - školitel (instructor) - integer foreign key
  - místnost (room) - integer foreign key
  - název, kategorie, kapacita a max. kapacita - stejně jako u šablony
    - slouží k přetížení pro konkrétní termín
- na frontendu
  - zobrazit (show_at) - timestamp
  - skrýt (hide_at) - timestamp
  - uzamknout (lock_at) - timestamp
  - notifikace (notify_at) - timestamp

== Školitel (instructor) ==
- na frontendu
  - jméno (name) - string
  - obrázek (image) - integer foreign key
  - HTML popis (description) - HTML string

== Místnost (room) ==
- na frontendu
  - název (name) - string
- zbytek doplní EKK podle kongresu

== Emailová šablona (email_template) ==
- na frontendu
  - název (name) - string
  - předmět (subject) - string
  - zpráva (body) - HTML string

== Email pro kurz (email) ==
- na frontendu
  - emailová šablona (template) - integer foreign key
  - kurz (event) - integer foreign key
  - událost, kdy se má odeslat (send_upon) - integer foreign key, nebo enum
  - předmět a zpráva - stejně jako u šablony
    - slouží k přetížení pro konkrétní email

== Obrázek (image) ==
- na frontendu
  - název (name) - string
  - mime type (mime_type) - string
  - data (data) - base64 string

== Klient (client) ==
- na backendu
  - číslo klienta (client_id) - string
  - číslo karty (card_id) - string
  - jméno (firstname) - string
  - příjmení (surname) - string
  - email (email) - string
  - status (status) - char
  - komentář (comment) - string
  - datum expirace (expiration_date) - date
  - zasílat zprávy (news) - boolean
  - osobní údaje (personal) - JSON s klíči sex, birth_date, phone, phone2
  - adresa (address) - JSON s klíči city, area_code, street, house_number
  - adresa 2 (address2) - stejně
  - firma (company) - JSON s klíči name, ico, dic

== Objednávka (order) ==
- needituje se přímo
  - id klienta (client) - integer foreign key
  - stav zaplacení (state) - string
  - číslo účtu (bill_number) - integer
  - id účtu (bill_ds_ident) - integer foreign key
  - identifikátor platby z platební brány (payment_ident) - string
  - id platby v rámci účtu (payment_ds_ident) - integer foreign key
  - id původní objednávky u storna (original_order_id) - integer foreign key
  - fiskalni kod (fik) - string
  - bezpečnostní kód poplatníka (bkp) - string
  - kdy odesláno na EET (sent_at) - timestamp

== Registrace (registration) ==
- needituje se přímo
  - kurz (event) - integer foreign key
  - objednávka (order) - integer foreign key
  - počet (count) - integer
  - komentář (comment) - string
  - potvrzeno (confirmed) - boolean

== Předregistrace (preregistration) ==
- needituje se přímo
  - kurz (event) - integer foreign key
  - klient (client) - integer foreign key
  - počet (count) - integer

== Slevový kupon (není, zatím neřešit) ==
- číslo
- typ (cizí klíč, nebo výčet?)
- stav (cizí klíč, nebo výčet?)
- sleva (jaké hodnoty má podporovat?)
- platnost
  - jaké hodnoty má podporovat?
  - výčet, nebo celočíselný počet měsíců?
- datum vytvoření
  - stačí jako začátek pro určení platnosti, nebo potřebujeme ješte sloupec na datum vydání klientovi?

- k obrázkům bude kromě klasických CRUD rout potřeba ještě GET routa, která bude jako response
  vracet přímo data se správným mime type
- ke každému modelu, se kterým může být spojeno více obrázků, bude potřeba navíc join-table
  - určit kde má být možnost více a kde stačí jeden
  - z dema soudím, že vedoucí má jeden, šablona kurzu může mít více
