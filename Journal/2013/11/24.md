https://github.com/brainspec/enumerize - Enumerated attributes with I18n and ActiveRecord/Mongoid support
'''
class User < ActiveRecord::Base
  extend Enumerize

  enumerize :sex, in: [:male, :female], default: lambda { |user| SexIdentifier.sex_for_name(user.name).to_sym }

  enumerize :role, in: [:user, :admin], default: :user
end
'''
* ''@user.sex_text # or @user.sex.text''
* ''User.sex.values # or User.enumerized_attributes[:sex].values''
'''
<%= form_for @user do |f| %>
  <%= f.select :sex, User.sex.options %>
<% end %>
'''
* ''user.sex = :male''
* ''user.sex.male? #=> true''
* ''user.sex.female? #=> false''
* when predicates: true option used in enumerize call, defines predicates directly on model (with possible prefix)
* ''enumerize :role, in: {:user => 1, :admin => 2}'' - store values in custom format
* can create scopes
