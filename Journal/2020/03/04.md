== instalace foodie na windows ==

- nainstalovat docker (https://hub.docker.com/editions/community/docker-ce-desktop-windows/)
- nainstalovat tiskarnu, nastavit ji jako sdilenou s rozumnym nazvem
- spustit RawPrintServer.exe STANDALONE "printer_name"
- v docker-compose.yml upravit PRINTER_HOST, PRINTER_PORT, EET_URL,
  prip. lze nastavit v konzoli pres set
- v konzoli:
docker login                       # login do public docker hubu - je treba ucet pro alto
docker login hub.alto-apps.mine.nu # login do naseho docker hubu - je treba ucet pro alto
set PRINTER_HOST=192.168.1.83
set PRINTER_PORT=9100
set EET_URL=http://192.168.1.97:8001
# ve slozce s docker-compose.yml
docker-compose.exe up
